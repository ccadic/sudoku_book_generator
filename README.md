Générateur de Livre Sudoku en Python

Projet Python permettant de générer automatiquement un livre complet de Sudoku au format Word (.docx).

Le script crée :

des centaines de grilles Sudoku valides,
plusieurs niveaux de difficulté (facile, moyen, difficile),
des pages prêtes à imprimer,
une mise en page automatique avec images des grilles,
une page de titre et des règles du jeu.

Fonctionnalités :

génération procédurale des grilles,
export DOCX compatible Microsoft Word,
création automatique des images Sudoku,
jusqu’à 1200 Sudoku par livre,
entièrement personnalisable.

Technologies utilisées :

Python
NumPy
Pillow (PIL)
python-docx

Idéal pour :

créer des livres Amazon KDP,
générer des cahiers d’activités,
apprendre les algorithmes de backtracking,
projets éducatifs Python.
